## 〇、开发环境搭建（VS Code + DeepSeek）

### 0.1 下载安装 VS Code
```
① 打开 https://code.visualstudio.com/
② 点击 Download for Mac（macOS 版本）
③ 双击下载的 .zip 文件
④ 把 VS Code 拖到 Applications（应用程序）文件夹
```

### 0.2 设置 VS Code 中文界面（可选）
```
① 打开 VS Code
② 左边栏点 Extensions（方块图标）
③ 搜索 "Chinese"
④ 安装 "Chinese (Simplified) Language Pack"
⑤ 右下角点 Restart 重启
```

### 0.3 连接 DeepSeek V4 模型
```
在 VS Code 中，直接打开右下角的 Copilot 聊天面板
→ 选择模型为 DeepSeek V4 Flash
→ 就可以开始对话和写代码了

注：这是 VS Code 内置的 GitHub Copilot 功能
   使用 DeepSeek 模型作为底层能力
   不需要自己申请 API Key
```

### 0.4 基础操作
```
☆ Command + J    → 打开/关闭终端
☆ Command + P    → 快速搜索文件
☆ Command + S    → 保存文件
☆ Command + N    → 新建文件
☆ 右键 → 在终端中打开 → 进入当前文件夹
☆ 拖拽 .html 文件到浏览器 → 预览网页
```

### 0.5 常见问题
```
Q: 国内连 GitHub 慢怎么办？
A: 代码上传用浏览器拖拽方式，比命令行稳定

Q: 聊天框不见了？
A: 点右上角 Copilot 图标重新打开

Q: 下载 VS Code 慢？
A: 可以从国内镜像站下载，或者耐心等待
```

## 二、GitHub 相关

### 2.1 注册 GitHub 账号
```
① 打开 github.com → Sign up
② 输入邮箱、密码、用户名
③ 邮箱验证
```

### 2.2 创建仓库（Repository）
```
① 点 + → New repository
② 仓库名：ai-fortune
③ 选 Public（公开）
④ 点 Create repository
```

### 2.3 上传代码到 GitHub
```
方式一：网页上传（推荐，国内网络友好）
  ① 打开仓库 → Upload files
  ② 拖入 index.html
  ③ 点 Commit changes

方式二：命令上传（国内网络可能报错）
  cd AI运势
  git init
  git add index.html
  git commit -m "说明"
  git remote add origin 仓库地址
  git push -u origin main
```

### 2.4 开启 GitHub Pages（部署网站）
```
① 仓库 → Settings → Pages
② Source → Deploy from a branch
③ Branch → main → / (root) → Save
④ 等几分钟，网站上线
```

## 三、域名相关

### 3.1 购买域名（阿里云）
```
① 打开 wanwang.aliyun.com
② 搜索想买的域名
③ 加入购物车 → 付款
④ 重要：域名实名认证（上传身份证）
   → 阿里云APP → 管控 → 域名 → 实名认证
   → 审核需 3-5 天
```

### 3.2 添加 DNS 解析
```
① 阿里云APP → 云解析 DNS
② 点击你的域名 → 添加记录：
   
   记录类型：CNAME
   主机记录：@ （代表主域名）
   记录值：你的用户名.github.io   （例：kittygreat11-cmyk.github.io）
   
   再添加一条：
   记录类型：CNAME
   主机记录：www
   记录值：你的用户名.github.io
   
TTL：保持默认 600（10分钟）
```

### 3.3 绑定域名到 GitHub Pages
```
① GitHub 仓库 → Settings → Pages
② Custom domain 输入：你的域名 → Save
③ 等 DNS 生效（几分钟到几小时）
④ 生效后访问：http://你的域名
```

### 3.4 SSL/HTTPS（自动配置）
```
GitHub Pages 会自动发 Let's Encrypt 证书
不需要在阿里云做任何操作，等即可
```

## 四、域名相关知识点

### 4.1 ICP 备案
```
国内服务器才需要备案（7-20天）
GitHub Pages 在国外，不需要备案
以后用国内服务器再备案
```

### 4.2 域名交易
```
注册后有 60 天锁定期，之后可以卖
通过阿里云一口价或经纪交易
```

### 4.3 一个域名挂多个项目
```
主域名 → 主站
子域名 → 不同项目（blog.xxx.com、tool.xxx.com）
只需子域名 + 不同 DNS 记录即可
```
